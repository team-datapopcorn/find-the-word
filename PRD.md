# 📄 [최종 PRD] 나만의 인터랙티브 Find the Word 퍼즐 생성기

## 1. 서비스 개요
- **서비스 명**: 퍼즐 메이커 (가칭)
- **목적**: 사용자가 직접 지정한 단어로 단어 찾기 퍼즐을 생성하고, 타이머 기능이 포함된 온라인 플레이 링크를 공유하여 이벤트용으로 활용.
- **핵심 가치**: *"세상에 하나뿐인 커스텀 퍼즐을 게임처럼 즐기기"*

---

## 2. 핵심 기능 (MVP Scope)

### 🧩 검수형 퍼즐 생성 엔진
- **입력**: 제목, 단어(최대 10개), 성공 메시지 입력.
- **프리뷰 및 재생성**: 생성된 퍼즐을 미리보고, 배치가 마음에 들지 않으면 `Regenerate(재생성)` 기능 제공.

### 🎯 좌표 기반 정답 판정 (Strict Validation)
- 랜덤하게 생성된 가짜 단어가 아닌, 생성 시 확정된 **정답 좌표**와 일치해야만 정답 처리.

### ⏱️ 타이머 플레이어
- `시작` 버튼 클릭 시 타이머 가동.
- 모든 단어의 좌표를 찾아내면 타이머 정지 및 기록 저장.

### 🔗 공유 및 히스토리
- 고유 URL 공유 기능.
- 내가 만든 퍼즐 목록 보관 기능.

---

## 3. 유저 플로우 (User Flow)

### 🛠️ 제작자 (나)
1.  **설정**: 퍼즐 제목, 단어 10개, 클리어 문구 입력.
2.  **검수**: 생성된 퍼즐 판 확인. (단어가 너무 뭉쳐있거나 가짜 단어가 보이면 `다시 생성` 클릭)
3.  **확정**: 배치가 마음에 들면 `퍼즐 발행` 클릭 → 고유 URL 생성.
4.  **배포**: URL 또는 QR 코드를 친구에게 공유.

### 🎮 플레이어 (친구/지인)
1.  **입장**: 링크 접속 → 제목과 단어 목록 확인.
2.  **플레이**: `시작` 클릭 후 드래그하여 단어 찾기.
3.  **완료**: 모든 단어 발견 시 기록(초)과 제작자의 메시지 확인.

---

## 4. 화면별 상세 UI 요소

### ① 퍼즐 생성 페이지 (Admin)
- **입력 폼**: 제목(`Text`), 단어 10개(`List`), 완료 메시지(`TextArea`).
- **프리뷰 영역**: 실시간으로 배치된 퍼즐 격자 미리보기.
- **제어 버튼**: `다시 생성(Shuffle)`, `퍼즐 발행(Publish)`.

### ② 퍼즐 플레이 페이지 (Player)
- **상단**: 타이머(`00:00`), 찾은 단어 개수(`N/10`).
- **중앙**: 퍼즐 그리드 (터치/드래그 인터랙션).
- **하단**: 찾아야 할 단어 리스트 (찾은 단어는 취소선 처리).
- **팝업**: "축하합니다! [00초] 기록!" + 제작자의 시크릿 메시지.

---

## 5. 예외 처리 및 논리 정책
- **배치 로직**: 단어는 가로, 세로, 대각선(옵션)으로 배치 가능하며, 단어 길이는 격자 크기를 초과할 수 없음.
- **정답 판정**: 플레이어가 드래그한 영역의 **시작 좌표**와 **끝 좌표**가 저장된 정답 데이터와 일치할 때만 성공으로 간주. (의도치 않게 생성된 단어 클릭 시 반응 없음)
- **중도 이탈**: 플레이 중 페이지 새로고침 시 기록은 초기화됨(재도전 유도).
- **수정 불가**: 한 번 발행된(URL이 생성된) 퍼즐은 수정할 수 없으며, 수정 필요 시 새로 생성해야 함.

## 6. 향후 추가 기능 (Backlog)

- **🏆 랭킹 보드**: 해당 링크를 타고 들어온 사람들 간의 시간 기록 순위표.
- **🎨 테마 스킨**: 시즌별(크리스마스, 할로윈 등) 배경 스킨 적용.

---

## 7. 기술 스택 제안 (추천)
- **Frontend**: `React` 또는 `Vue` (드래그 인터랙션 구현 용이)
- **Backend/Storage**: `n8n` (데이터 처리) + `Supabase` (데이터베이스 및 호스팅)
- **Algorithm**: 단어 배치 시 `Random Position Assignment` 알고리즘 활용.
